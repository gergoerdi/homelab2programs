# sokoban
A tradicionális Sokoban program 1 visszalépési (undo) lehetőséggel, 60 pályával. ( A pályákért köszönet begoon-nak: https://github.com/begoon/sokoban-maps )
A sokoban.bas a BASIC forráskód, amiben a program íródott.
A sokoban.final.BAS az a forráskód, ami végül betöltésre is kerül.
A htp fájl emulátorokhoz, a wav fájl valódi gépekhez használható.
